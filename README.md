# 📄 Smart Resume Analyzer
### CSE — Machine Learning Lab Project

> An end-to-end NLP pipeline that classifies resumes into job roles using TF-IDF feature extraction and a Soft Voting Ensemble of machine learning models, deployed via a Streamlit web application.

---

## 👥 Author:
- Lavanya Sharma

---

## 🎯 Problem Statement
Given the raw text of a resume, predict which of **25 job categories** it belongs to — such as Data Science, Java Developer, DevOps Engineer, HR, Web Designer, and more.

---

## 📊 Dataset
**Updated Resume Dataset** — [Kaggle Link](https://www.kaggle.com/datasets/jillanisofttech/updated-resume-dataset)

| Attribute | Value |
|-----------|-------|
| Total Samples | 2,484 |
| Job Categories | 25 |
| Format | CSV (text + label) |
| Avg. Words per Resume | ~200 |

## 🤖 Models Used

| Model | Role |
|-------|------|
| Logistic Regression | Strong linear baseline for sparse TF-IDF |
| Random Forest | Non-linear ensemble, captures feature interactions |
| LinearSVC (Calibrated) | Best single model for text classification |
| MLP Neural Network | 2-layer neural net (512→256), learns non-linear patterns |
| **Soft Voting Ensemble** ⭐ | Averages probabilities from LR + SVC + MLP — **best accuracy** |

---

## 📈 Pipeline

```
CSV Dataset
    ↓
Exploratory Data Analysis (EDA)
    ↓
Text Preprocessing (URL/email removal, stopwords, lowercase)
    ↓
TF-IDF Vectorisation (50,000 features, unigrams + bigrams)
    ↓
Train / Test Split (80/20 stratified)
    ↓
5 Models trained with GridSearchCV tuning
    ↓
5-Fold Stratified Cross-Validation
    ↓
Soft Voting Ensemble (best model saved)
    ↓
Streamlit App — PDF upload → job role prediction
```

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/smart-resume-analyzer.git
cd smart-resume-analyzer
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download NLTK data
```bash
python3 -c "import nltk; nltk.download('stopwords'); nltk.download('punkt')"
```

### 4. Generate the model files
- Open `Smart_Resume_Analyzer.ipynb` in Google Colab
- Upload `UpdatedResumeDataSet.csv` when prompted
- Run all cells (Runtime → Run All)
- Download `model.pkl`, `tfidf.pkl`, `label_encoder.pkl` from Section 10
- Place them in the project folder

### 5. Run the app
```bash
streamlit run app.py
```
Opens at **http://localhost:8501**

## 📋 Evaluation Metrics

| Metric | Description |
|--------|-------------|
| Accuracy | Overall correct predictions |
| Weighted F1 | F1 averaged by class support |
| MCC | Matthews Correlation Coefficient — reliable for multi-class |
| 5-Fold CV | Cross-validated accuracy with std deviation |
